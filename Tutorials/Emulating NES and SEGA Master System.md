## Emulating NES and SEGA Master System on EZ-Flash  
### Do I need a computer?

**Short answer: Yes, you need a computer (PC, Mac, or Linux) for initial setup.**

---

## Why a Computer Is Required

NES and SEGA Master System games **do not run natively** on the Game Boy Advance.  
To play them on an **EZ-Flash**, you must:

- Use a **GBA emulator wrapper** (PocketNES or SMSAdvance)
- **Convert or bundle ROMs** into a `.gba` file
- Copy the finished files to the EZ-Flash microSD card

All of that conversion work **must be done on a computer**.

Once set up, you **do not need the computer again** unless you want to add or change games.

---

## What Emulators Are Used

- **NES**: PocketNES  
- **SEGA Master System / Game Gear**: SMSAdvance or SMSAdvance X  

These emulators run *inside* the GBA as normal `.gba` games.

---

## Basic Workflow

1. Download the emulator (PocketNES or SMSAdvance)
2. Use the included **builder tool** on your computer
3. Add your `.nes` or `.sms` ROMs
4. Build a `.gba` file
5. Copy the `.gba` file to the EZ-Flash microSD
6. Launch it on real hardware

## TL;DR

- You do need a computer for the initial setup.
- NES and SMS are software emulated.
- You cannot add raw `.nes` or `.sms` files directly on the GBA.
- After setup, everything runs directly from the EZ-Flash.
